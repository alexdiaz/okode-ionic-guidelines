# Ionic project and code style guidelines

## Introduction
This is the style guide defined by Okode for the development of apps with Ionic / Angular.

## Project layout

```ruby
└── src
    ├── app

    │   ├── app.html
    │   ├── app.module.ts
    │   └── main.ts
    ├── assets
    ├── components    <!-- generator -->
    ├── directives    # generator
    ├── pages         \\ generator
    ├── pipes         \\ generator
    ├── models
    ├── services      \\ generator
    ├── theme
    ├── utils
    └── index.html
```
